# richman

💳 Richman — 스마트 금융 통합 플랫폼

소비 분석 · 주식 · 크립토를 하나의 흐름으로 연결하고, AI 챗봇이 금융 생활 전반을 해석해주는 스마트 금융 플랫폼


🧭 서비스 소개
Richman은 단순 가계부나 시세 조회 앱이 아닙니다.
사용자의 실제 금융 데이터를 통합 관리하고, AI가 개인화된 인사이트를 제공합니다.
소비 데이터 수집 → 패턴 분석 → 주식/크립토 조회 → 감성 분석 → AI 챗봇 인사이트

✨ 핵심 기능
🧾 소비 분석

소비 캘린더 — 날짜별 지출 시각화, 무지출 데이 배지
소비 데이터 분석 — 카테고리별 도넛 차트, 고정 지출 추적, AI 소비 리포트
스마트 정산 — N빵 정산 등록 후 실제 소비에 자동 반영

📈 주식

종목 검색 & 자동완성 — 국내(KRX) · 해외(NASDAQ, NYSE) 지원
기술 지표 시각화 — 이동평균선(MA), 볼린저 밴드 차트 오버레이
관심 종목 & 포트폴리오 — 보유 수량/단가 입력 → 수익률 자동 계산
AI 예측 히스토리 — 예측가 vs 실제 종가 비교, 오차 지표(MAE·RMSE) 공개

🪙 크립토

실시간 시세 대시보드 — Upbit API 기반 현재가 · 등락률 · 거래량
Social Buzz 지표 — 뉴스 언급 빈도 + 거래량으로 코인 관심도 수치화
감정 지표 — GPT 기반 뉴스 감성 분석 (긍정/중립/부정)

🤖 AI 통합 챗봇
사용자의 실제 데이터를 컨텍스트로 주입해 개인화된 금융 인사이트 제공
질문 유형예시 답변소비"이번 달 배달 앱 지출이 전월 대비 34% 증가했어요."주식"삼성전자는 현재 20일 이평선 아래, 뉴스 감성 중립(48점)입니다."크립토"BTC 뉴스 언급량 40% 증가, Buzz Score 72점으로 높은 편입니다."복합"이번 달 총 소비 1,240,000원, 주식 +3.2%, BTC -1.4%, 여유 자금 약 380,000원."

⚠️ 모든 투자 관련 답변에는 "이 정보는 참고용이며 투자 손실에 대한 책임을 지지 않습니다" 문구가 자동 포함됩니다.
